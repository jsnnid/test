一周院线新片

《21克拉》

上映时间：2018年4月20日

类型：喜剧 / 爱情

导演：何念

编剧：马天 / 李扬 / 何念

演员：郭京飞 / 迪丽热巴 / 大鹏 / 包贝尔

期待值：3星

何念，上海话剧圈知名卖座导演。这部改编自其同名话剧《21克拉》的电影，曾巡演上百场，口碑相当不错，甚至谢娜、何炅都排演过一版，在湖南卫视的《星剧社》播出。

电影男主依然是话剧首版的男主郭京飞饰演，这乃是一大惊喜，也是一大挑战。话剧版郭京飞夸张的演绎放在电影中未必适用，这些年演了各种款男士的郭京飞，再次在大银幕演回当年那个“抠抠男”，会是怎样的表现？

而女主角则由近来一直很火的迪丽热巴担任，天生丽质的热巴饰演一位爱包包如命的“拜金女”，和热巴走红的那部电视剧《克拉恋人》的形象有些相似，只不过如今的热巴已摇身一变成女主角。

何念早些年导演过一些电视剧，最有名的就是2010年姚晨和凌潇肃主演的那部《和空姐在一起的日子》。那么这部一样非常有都市情感气息的爱情喜剧，是否会再次成为爆款？拭目以待。

《脱单告急》

上映时间：2018年4月20日

类型：动作 / 科幻 / 冒险

导演：任鹏

编剧：瑞安·恩格尔 / 卡尔顿·库斯

演员：董子健 / 钟楚曦 / 春夏 / 袁福福

期待值：3星半

春天可能真的比较适合恋爱，同一天两部粉红系影片上映，相比《21克拉》的都市喜剧风，这部《脱单告急》则显得是校园青春风，从主创阵容看，还有些文艺调调，董子健＋钟楚曦＋春夏，这是集齐了金马、金像的最佳新人啊。

故事讲述的是一个校园“大姐大”（钟楚曦 饰）教单身四年的男青年（董子健 饰）追校园女神（春夏 饰），研制了一套所谓的“恋爱公式”，但是熟悉套路的我们知道，公式肯定是不管用的。

这一类电影的看点就在于“恋爱公式”的花样以及这段男女主人公最后认清彼此真心、打破套路走到一起的方式。无论如何，能看三位未来的影帝、影后在青春岁月的青春戏，应该是很难得的吧。

《犬之岛》

上映时间：2018年4月20日

类型：喜剧 / 动画 / 冒险

导演：韦斯·安德森

编剧：韦斯·安德森 / 罗曼·科波拉 / 詹森·舒瓦兹曼 / 野村训市

演员：布莱恩·科兰斯顿 / 爱德华·诺顿 / 科宇·兰金

期待值：4星

柏林电影节时候就听闻了这部“怪才”导演韦斯·安德森的新作，今年的北京国际电影节也请到了该片做超前点映，电影票都一售而空。好在不几天大银幕就都可以看到，让影迷们可以过足瘾。

在《了不起的狐狸爸爸》后，安德森再次做定格动画，只不过这一次主角变成了狗狗，一群说着人话的狗狗，背后隐喻的是日本的社会文化。

电影中延续了安德森式的幽默，怪诞中带着暗黑系。影片的色彩与配乐都非常好看，章节式的多线叙事，加上符号化、政治指涉明显的隐喻，成为了这部影片最为人们津津乐道的看点之一。

故事背景设置在日本长崎，浮世绘风格的影片开头描述了一个城市被狗掌控的样貌。狗狗重回巅峰，而人类成为了被驯服的对象。然而没过多久，由于“犬流感”爆发，市长把所有的狗都驱逐到一座小岛上，这个被命名为“犬之岛” 的地方，上演了一出以狗狗视角展开的故事。

电影配音邀请了众多大咖，其中之一便是刚刚凭借《三块广告牌》获得最佳女主的弗兰西斯·麦克多蒙德，还有我们的“黑寡妇”斯嘉丽·约翰逊等等。

怎么解读影片背后的隐喻或是其次，最重要的是终于能在锅内大银幕上观看这部作品，还给安德森一张电影票。

《米花之味》

上映时间：2018年4月20日

类型：剧情

导演：鹏飞

编剧：鹏飞 / 英泽

演员：英泽 / 叶不勒

期待值：4星

这部国产片最早在去年平遥国际电影节中展映，当时获得了新生代单元最受欢迎影片。今年在北京国际电影节国片单元也有展映，作为今年几部导演新作之一，可以说还是非常独具风味并且值得关注的一部新作品。

影片关注的是留守儿童。但导演并未刻意地去表达或是批判，而是亲自去体验，在体验中完成了“日常化”的剧本，再让演员去演绎。

导演的影像表达非常值得我们去解析，是一种客观记录生活变迁却又在贫瘠生活表面下，彰显出盎然生机的感觉。导演在采访中多次提及是枝裕和，或是从是枝裕和那里汲取到的养分，如何自然而不做作的构建出有戏剧性的人物关系，再加上非自然的光影却营造出现实感的方式，是一种新的现实主义美学表达。

无论是从主题还是从新生代美学方面，这都是一部值得我们关注的国产片。希望能在院线上映时间长一些。

一周网络新片

《方寸之爱》

类型：喜剧

导演：Anand Tiwari

编剧：Asif Ali Zardari

演员：维杰·卡沙尔 / Angira Dhar / Alankrita Sahai

推荐值：4星

印度电影不仅是在中国越来越火，整个世界都盯上了宝莱坞丰富的原创力。这部电影由Netflix和本土公司UTV 影业共同打造的影片，是印度首部专门为流媒体打造的电影。

电影讲述的是全球难题——年轻人买房。就买房加上爱情故事，再配上印度电影的宗教话题，整个故事让人看得津津有味。

2个多小时的电影，音乐和歌舞占据了大部分。尽管有了美国资本介入，但是印度味百分百还原，宗教问题、社会阶层问题还有大家熟悉的印度音乐和歌舞，尽管长，但是足够让人笑中带泪，而影片的光影和剪辑并没有因为是在网络媒介播出而粗糙，一样好看，当然也更加符合好莱坞审美。

《淹没》

类型: 剧情 / 爱情 / 惊悚

导演: 维姆·文德斯

编剧: 艾琳·迪格纳穆 / JM Ledgard

主演: 詹姆斯·麦卡沃伊 / 艾丽西亚·维坎德 / 亚历山大·希迪格

推荐值：4星

这部文德斯的新作在圣塞巴斯蒂安国际电影节作为开幕片放映。作为“德国新电影四杰”之一的文德斯，是和伍迪·艾伦、戈达尔一样多产且活到老拍到老的电影大师。而这部商业性略强的电影，总让人想起文德斯自己1991年《直到世界尽头》的感觉，也会有网友觉得，这部影片应该与赫尔佐格的盐与火对比来看。两位大师在新时期带有商业诉求的艺术电影，一定程度上牺牲了艺术的神秘性，去注重叙事，是一种妥协还是一种融合，值得玩味。

故事围绕女科学家丹妮尔与英国商人詹姆斯的爱情故事展开。詹姆斯因被伊斯兰圣战分子囚禁，与丹妮尔远隔千里，丹尼尔需要想办法拯救自己的爱人。

无论如何，这部电影有一美还有坎妹两位高颜值演员加盟，光是看他们的侧颜，也心满意足了。
